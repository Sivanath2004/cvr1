AIM:
To perform basic image operations such as loading, displaying, color format conversion, and image
enhancement.
Procedure:
Step-1 Load an image using OpenCV or PIL.
Step-2 Display the image using a library like OpenCV or Matplotlib.
Step-3 Convert the image between different color formats (RGB, Grayscale, HSV).
Step-4 Apply contrast and brightness adjustments for image enhancement.
Step-5 Save the processed image.
